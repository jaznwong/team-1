TEAM 1 - Central Park Conservancy!
How To Setup:
- Backend:
1. Install postgresql on your system
2. Create user: cpc_user/password: cpc_user/database: cpc_data
3. cd into Backend
4. Run npm install
5. Run npm start
6. Continue to run node/express erver in background

- Frontend:
1. cd in my-app
2. Run npm install
3. Run npm start
